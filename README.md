### SpringCloud 概览    

-----------------------
|  模块   | 可用组件  |
|  :----  | :----  |
| 服务注册与发现  | Eureka、Zookeeper、Consul |
| 服务调用负载均衡  | Ribbon、Feign |
| 服务容错保护  | Hystrix |
| 断路器监控  | HystrixDashboard |
| API网关  | Zuul、SpringCloudGateWay |
| 外部集中化配置管理  | 单元格 |
| 消息总线  | SpringCloudBus |
-------------------------
